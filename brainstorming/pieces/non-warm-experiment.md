# Non-warm experiment

VISION says warm-only. Is that rule true, or is it just the rule I've
settled into because every cold-palette experiment I've tried felt
dishonest? I haven't actually *finished* a cold-palette piece.

## The test

Build one deliberate piece in a non-warm palette. Not to rebel — to
interrogate. Success condition: *I* like the result better than the
warm version would have been. If I don't, the warm-only rule is
confirmed. If I do, update VISION.

## Candidate palettes to try

- **Steel / cobalt / ember** — cold with a single warm accent. Common in
  techno / industrial visuals. Risk: kitschy.
- **Deep teal / midnight / moss** — cool but dark-biased. Melancholy.
  Could work for a slow piece with long decays.
- **Monochrome warm-blue** — a single hue cycled from near-black to
  pale cream through dark blue. Tests whether it's *warm* I care about
  or *warmth-feel* (tonal warmth via saturation + luminance).

## Likely best match

A **reaction-diffusion or flow-field piece in steel/cobalt**. The
technique is itself cold/mechanical-feeling; warm colours would fight
that character. Let the palette match the form's intrinsic temperature.

## Guard rail

Still: no complementary-hue jumps. No pink-next-to-green. Even if the
base palette is cold, transitions stay within the palette's family.
The warm-only rule is really a *cohesion* rule disguised as a hue rule.
